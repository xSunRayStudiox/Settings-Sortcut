Accessing Application Settings

  1. Select an App: From the list of apps, select the app you want to manage.
  2. App Info: You will see the App Info screen with various options to manage the app settings. The options typically include:
  • Disable: This option will disable the app, preventing it from running and appearing in the app drawer. Useful for pre-installed apps that cannot be uninstalled.
  • Force Stop: Stops the app from running. Use this if an app is unresponsive or behaving abnormally.
  • Notifications: Customize how the app sends notifications, including turning them off entirely.
  • Permissions: Adjust the permissions granted to the app, such as access to the camera, location, or contacts.
  • Storage & Cache: Manage the app's storage by clearing its cache or data.
  • Screen Time: View how much time you have spent using the app.
  • Battery & Performance: Check the app's battery usage and manage its background activity to save battery life.
  • Open by Default: Set or clear default actions for the app.

Additional Management Options

  1. Special App Access: In the "Apps & notifications" menu, select "Advanced" and then "Special app access" to manage specific permissions like battery optimization, device admin apps, and more.
  2. Default Apps: Set default apps for actions like browsing, messaging, and calling by selecting "Default apps" in the "Apps & notifications" menu.
  3. App Permissions: In the "Apps & notifications" menu, select "Permissions manager" to manage permissions for all apps from one place.
Note: Changing app settings can affect how apps behave on your device. Make sure to understand the impact of each setting before making changes.
