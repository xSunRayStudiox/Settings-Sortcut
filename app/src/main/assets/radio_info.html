Dialer Method:

  1. Open the phone's dialer app.
  2. Enter the following code: ##4636##
  3. A hidden menu will appear. Select 'Phone information' to access the RadioInfo screen.

Direct Access (if supported by your Device):

  1. Open your app and navigate to the RadioInfo section.
  2. Follow the in-app instructions to access the RadioInfo screen.

How to use:

  • Turn Radio On/Off: Enables or disables the phone's radio.
  • Select Phone Index:
    Phone 0 = SIM 1
    Phone 1 = SIM 2
  • Set Preferred Network Type: Allows you to choose the preferred network type.
  • Some Network Type
    LTE only = 4G
    NR only = 5G
  • Run Ping Test: Tests connectivity to the network.

Additional Information:

  • Neighboring Cell Info: Displays information about neighboring cell towers.
  • Data Service and Activity: Shows the status of data services and their activity.
  • SMS Service Center Number: Displays the number of the SMS service center.
