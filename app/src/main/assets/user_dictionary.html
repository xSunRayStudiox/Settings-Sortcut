Accessing User Dictionary Settings

  1. Open Settings: Start by opening the Settings app on your Android device.
  2. Navigate to System: Scroll down and select "System."
  3. Language & Input: Tap on "Language & input."
  4. Personal Dictionary: Select "Personal dictionary" or "User dictionary" to access the user dictionary settings.

Adding a New Word

  1. Add Word: Tap on the "+" or "Add" button to add a new word to the dictionary.
  2. Enter Word: Type the word you want to add to the dictionary.
  3. Shortcut (Optional): You can also add a shortcut for the word, which will allow you to type the shortcut and have it automatically replaced by the full word.
  4. Save: Tap "Save" or "Done" to add the word to your personal dictionary.

Editing or Deleting a Word

  1. Select Word: In the personal dictionary, tap on the word you want to edit or delete.
  2. Edit Word: To edit, modify the word or shortcut and save the changes.
  3. Delete Word: To delete, tap on the delete icon (usually a trash can) to remove the word from your dictionary.

Using the User Dictionary

  1. Typing: When you type a word that is in your personal dictionary, your keyboard will suggest it as you type.
  2. Shortcuts: If you added shortcuts, typing the shortcut will automatically replace it with the full word, making typing quicker and easier.
Note: The user dictionary helps improve typing efficiency by allowing you to add frequently used words and shortcuts. Customize it to suit your typing habits.
